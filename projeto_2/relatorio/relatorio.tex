\section{Experimentos}
Para esta seção vamos começar a olhar para uma implementação real dos nosso modelos de inteligência artificial:
Bellman e Q-Learning destrinchando as suas composições e alterando os valores das variáveis que regem o comportamento
dois dois de forma a tentar extrair o máximo de informção sobre o funcionamento, estratégia e rendimentos no fluxo de
decisão.

Para começar, vamos fazer um leve retrospecto no que concerne à modelagem matemática dos dois modelos:
\begin{itemize}
	\item Modelo de Bellman para value iteration (Modelo Planejador): O que o modelo de Bellman faz em poucas palavras é
	      tentar estar ciente de toda a matriz de probabilidade de transição entre os estados do nosso experimento de forma a
	      nos retornar a sequência de ações ótima (claramente dentro das capacidades estabelecidas pelas nossas variáveis)
	      \begin{equation}
		      V(s) = \max_{a} \left( R(s,a) + \gamma \sum_{s'} P(s'|s,a) V(s') \right)
	      \end{equation}
	\item Modelo Q-Learning (Aprendizado na raça): Cai de paraquedas no ambiente a ser estudado e vai aos poucos entendendo
	      como tomar as melhores decisões.
	      \begin{equation}
		      Q(s, a) = R(s, a) + \gamma \sum_{s'} P(s'|s,a) \max_{a'} Q(s', a')
	      \end{equation}
\end{itemize}
